Описание проекта
Проект представляет собой REST API, разработанное на Go, для управления данными о трех сущностях: автомобилях, мебели и цветах.
API позволяет добавлять новые записи для каждой сущности и получать список всех объектов.
Данные хранятся в памяти, без использования базы данных.

Подготовительные действия
git clone https://example.com/repository.git

Как запустить проект
Сервер будет доступен по адресу: http://localhost:8080
